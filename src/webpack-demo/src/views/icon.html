Icon 图标（含fontAwesomev4.5——ace已内置，glyphicon，自定义字体图标） 预览
字体图标 <FontAwesome>
基本
fa-asterisk

组件说明

包括600多个（fontAwesome4.7为675个）来自 Font Awesome 的字体图标。Font Awesome。

字体图标 <Glyphicons>
基本
glyphicon-asterisk

组件说明

包括263个来自 Glyphicon Halflings 的字体图标。Glyphicons Halflings 一般是收费的，但是他们的作者允许 Bootstrap 免费使用。为了表示感谢，希望你在使用时尽量为 Glyphicons 添加一个友情链接。

自定义svg转字体图标 <icon-font-generator>
基本
glyphicon-asterisk

组件说明

此示例展示包括75个来自自定义的字体图标。icon-font-generator通过易于使用，预配置的cli工具，可从一堆.svg文件生成webfont图标工具包。

安装
npm install -g icon-font-generator
快速使用
icon-font-generator my-icons/*.svg -o icon-dist
